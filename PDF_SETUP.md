# PDF 和圖片生成功能 - 依賴安裝指南

## 概述

電視牆製作功能目前提供了完整的模板，但需要安裝額外的依賴包才能實現實際的 PDF 和圖片生成功能。

## 必要依賴

為了啟用完整的 PDF 和圖片生成功能，您需要安裝以下依賴：

```bash
# 核心依賴
npm install jspdf html2canvas

# TypeScript 類型定義 (如果使用 TypeScript)
npm install @types/jspdf
```

## 依賴說明

### html2canvas

- **用途**: 將 HTML 元素轉換為 Canvas，然後生成圖片
- **版本**: 建議使用最新穩定版本
- **文檔**: https://html2canvas.hertzen.com/

### jsPDF

- **用途**: 在瀏覽器中生成 PDF 文件
- **版本**: 建議使用最新穩定版本
- **文檔**: https://github.com/parallax/jsPDF

## 啟用步驟

1. **安裝依賴**:

   ```bash
   npm install jspdf html2canvas @types/jspdf
   ```

2. **更新 pdfGenerator.ts**:
   取消註釋 `/src/utils/pdfGenerator.ts` 中的實際實現代碼，並刪除模擬實現。

3. **測試功能**:
   重新啟動開發服務器，前往電視牆製作頁面測試圖片和 PDF 生成功能。

## 使用建議

1. **開發階段**: 可以先使用模擬功能進行 UI 測試和流程驗證
2. **生產環境**: 務必安裝實際依賴以提供完整功能
3. **性能考量**: 生成高解析度圖片可能需要較長時間，請適當提示用戶

## 故障排除

### 常見問題

1. **圖片生成失敗**

   - 檢查是否已安裝 html2canvas
   - 確認沒有跨域問題 (CORS)
   - 檢查 HTML 元素是否可見

2. **PDF 生成問題**

   - 檢查是否已安裝 jsPDF
   - 確認圖片尺寸和 PDF 頁面尺寸匹配

3. **字體顯示問題**
   - 使用系統字體避免字體載入問題
   - 在生成前確保自訂字體已完全載入
